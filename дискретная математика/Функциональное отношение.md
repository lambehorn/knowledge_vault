Отношение F между [множествами](множество) A и B называется функциональным отношением (или просто функцией), если каждому элементу из множества A соответствует не более одного элемента [множества](множество) B.

[множества](множество) $D=\{a\in A|\exists b\in B:(a,b)\in F\}$  - называется областью определения функции.
[множества](множество) $E=\{b\in B|\exists a\in A:(a,b)\in F\}$ - называется областью значения функции.

Если каждому элементу из A соответствует ровно один элемент из B то есть D=A, то функция называется тотальной, иначе - частичной.
 Тотальные функции называют отображениями.
 
Тотальная функция F:A -> B -называется биективной, если она сюрьективна и иньективна.

Биективная функция - это отношение, которое:
1. Инъективно
2. Сюрьективно
3. Непустозначное
4. функциональное
