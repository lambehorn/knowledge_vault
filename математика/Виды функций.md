Основные виды [функций](функция) в [математике](математика):

### [[Элементарные функции]]
Это функции которые выражаются через конечное алгебраическое число алгебраических операций. Данный тип функций строится из простейших функций.
### [[Неэлементарные функции]]
Это функции которые не выражаются через элементарные функции с помощью конечного числа операций
