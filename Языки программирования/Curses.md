Это библиотека написаная для языка [[Python]], которая позволяет писать консольные приложения.

# Начало
для того что бы консольное приложение запустилось необходимо импортировать библиотеку curses. Так же можно импортировать обьект wrapper, для удобства, который позволит обрабатывать ваше приложение, что по сути является оберткой.

```python
import curses
from curses import wrapper
```

Действия происходят внутри функции main, в котороую wrapper помещает объект - текущее окно консоли, с которым и происходит основное взаимодействие.

```python
def main(stdscr): # Созддаем функцию с объектом окна консоли stdscr
	pass
wrapper(main) # Оборачиваем функцию в wrapper
```
