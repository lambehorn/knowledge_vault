[[Команда]] для сжатия, архивации или распаковки. 
tar - это архиватор, а не компрессор. Он собирает файлы в один контейнер, но сам по себе не сжимает (если не добавить опции).

### Основные опции tar:
- c - создать архив
- x - распаковать архив
- v - показывать процесс
- f - указывать имя архива
- z - сжимать/распаковывать через gzip (.tar.gz)
- j - через bzip2 (.tar.bz2)
- J - через xz (.tar.xz)
