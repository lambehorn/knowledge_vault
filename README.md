# Инструкция по установке

1. Скачать программу [Obsidian](https://obsidian.md/), или любой другой редактор для файлов .md.
 - Arch
```bash
sudo Pacman -S obsidian
```
- Debian\Ubuntu\Fedora\CentOS
```bash
sudo snap install obsidian --classic
```
2. скачать git [репозиторий](https://github.com/lambehorn/knowledge_vault.git).
```bash
git clone https://github.com/lambehorn/knowledge_vault.git
```
3. Открыть репозиторий при помощи текстового редактора который вы установили.

---

Конспект предназначен для учащихся на прикладном потоке ПГНИУ, но пользоваться им может каждый желающий.

Так же можно отправлять правки в репозиторий, что бы развивать конспект.

# Installation guide

1. Download the app [Obsidian](https://obsidian.md/) (or any other editor for `.md` files).
 - Arch
```bash
sudo pacman -S obsidian
```
- Debian  Ubuntu  Fedora  CentOS
```bash
sudo snap install obsidian --classic
```
2. Clone the git [repository](https://github.com/lambehorn/knowledge_vault.git):
```bash
git clone https://github.com/lambehorn/knowledge_vault.git
```
3. Open the cloned folder with the text editor you installed above.   

---

The notes are made primarily for students of the Applied-Stream at Perm State University (PGNIU), but anyone is welcome to use them.

Feel free to submit pull requests to help improve the notes.
